## SECTION 9: PREPARATION CHECKLIST

### Two Weeks Before Trial
- [ ] Organize all evidence into numbered exhibits
- [ ] Create exhibit list with descriptions
- [ ] Prepare witness list and secure their attendance
- [ ] Draft opening statement
- [ ] Draft closing argument
- [ ] Prepare direct examination questions
- [ ] Prepare cross-examination questions
- [ ] Calculate final damages amount
- [ ] Review all statutes and case law
- [ ] Create timeline chart/visual aids if helpful

### One Week Before Trial
- [ ] Practice opening statement (time it)
- [ ] Practice presenting each exhibit
- [ ] Practice closing argument (time it)
- [ ] Prepare copies: 3 sets of all exhibits (judge, defendant, your copy)
- [ ] Confirm all witnesses will attend
- [ ] Visit courtroom if possible to familiarize yourself
- [ ] Prepare professional attire
- [ ] Review opponent's likely arguments and your rebuttals

### Day Before Trial
- [ ] Review entire file one more time
- [ ] Get good sleep
- [ ] Prepare folder/binder with organized exhibits
- [ ] Print extra copies of key documents
- [ ] Prepare notepad for notes during trial
- [ ] Check court location and parking
- [ ] Set multiple alarms

### Day of Trial
- [ ] Arrive 30 minutes early
- [ ] Dress professionally
- [ ] Bring: All exhibits, witness list, notes, pen, water
- [ ] Turn off cell phone
- [ ] Remain calm and respectful at all times
- [ ] Speak clearly and directly to the judge
- [ ] Listen carefully to all questions
- [ ] Don't interrupt the judge or opposing party
