# Case Preparation Checklist - Jeff Mills

## Pre-Filing Preparation

### ✅ Evidence Collection
- [ ] Gather all photos and screenshots
  - [ ] Date and time stamp all images
  - [ ] Organize chronologically
  - [ ] Create captions explaining what each shows
  - [ ] Save originals AND make copies
  
- [ ] Collect all written communications
  - [ ] Emails (print or save as PDF)
  - [ ] Text messages (screenshot with dates/times)
  - [ ] Letters (copies, not originals)
  - [ ] Social media messages
  - [ ] Voicemail transcripts
  
- [ ] Gather all financial records
  - [ ] Rent receipts or cancelled checks
  - [ ] Security deposit receipt
  - [ ] Utility bills
  - [ ] Repair costs (receipts)
  - [ ] Moving costs (if applicable)
  - [ ] Hotel/temporary housing costs
  - [ ] Medical bills (if health affected)
  - [ ] Lost wages documentation
  
- [ ] Obtain official documents
  - [ ] Lease or rental agreement (original and all amendments)
  - [ ] Move-in inspection report
  - [ ] Health department inspection reports
  - [ ] Police reports (if applicable)
  - [ ] Building code violations
  - [ ] Prior court documents
  
- [ ] Create documentation logs
  - [ ] Phone call log (who, when, what discussed)
  - [ ] Incident log (date, time, what happened)
  - [ ] Witness interaction log
  - [ ] Repair request log

### ✅ Witness Preparation
- [ ] Identify all potential witnesses
  - [ ] Other tenants
  - [ ] Neighbors
  - [ ] Inspectors
  - [ ] Repair persons
  - [ ] Friends/family who witnessed issues
  
- [ ] Collect witness information
  - [ ] Full names
  - [ ] Current addresses
  - [ ] Phone numbers
  - [ ] Email addresses
  - [ ] Relationship to case
  
- [ ] Prepare witness statements
  - [ ] What they saw/heard
  - [ ] When they witnessed it
  - [ ] Signed and dated statements
  - [ ] Notarized if possible
  
- [ ] Confirm witness availability
  - [ ] Willing to testify?
  - [ ] Available on court date?
  - [ ] Need subpoena?

### ✅ Timeline Creation
- [ ] Create comprehensive timeline
  - [ ] Start date (move-in date)
  - [ ] First issue noticed
  - [ ] Each complaint made to landlord
  - [ ] Each landlord response (or lack thereof)
  - [ ] Health department contacts
  - [ ] Repairs made (or not made)
  - [ ] Any retaliation
  - [ ] Move-out date (if applicable)
  
- [ ] Include supporting evidence for each event
  - [ ] Photos
  - [ ] Emails
  - [ ] Receipts
  - [ ] Inspection reports

### ✅ Legal Research
- [ ] Identify applicable laws
  - [ ] Minnesota Statutes Chapter 504B (Landlord-Tenant)
  - [ ] Local housing codes
  - [ ] City ordinances
  - [ ] Specific violations
  
- [ ] Research similar cases
  - [ ] Court opinions
  - [ ] Verdict amounts
  - [ ] Legal precedents
  
- [ ] Understand legal elements of your claims
  - [ ] What you must prove
  - [ ] Burden of proof
  - [ ] Defenses defendant might raise
  
- [ ] Review Legal Jargon Glossary
  - [ ] Understand key terms
  - [ ] Know what judge will ask
  - [ ] Practice explaining your case

### ✅ Calculate Damages
- [ ] Itemize all economic damages
  - [ ] Overpaid rent (if uninhabitable)
  - [ ] Security deposit (if wrongfully withheld)
  - [ ] Moving costs
  - [ ] Storage costs
  - [ ] Hotel/temporary housing
  - [ ] Repair costs you paid
  - [ ] Damaged property
  - [ ] Lost wages
  - [ ] Medical expenses
  
- [ ] Document each damage with evidence
  - [ ] Receipts
  - [ ] Invoices
  - [ ] Estimates
  - [ ] Photos of damaged items
  
- [ ] Research potential statutory damages
  - [ ] Security deposit violations (penalties)
  - [ ] Retaliation (damages)
  - [ ] Code violations (fines you can claim)
  
- [ ] Total your damages
  - [ ] Add up all items
  - [ ] Ensure under $15,000 for conciliation court
  - [ ] Prepare detailed breakdown

### ✅ Venue Selection
- [ ] Review Filing Options Guide
- [ ] Determine primary goal
  - [ ] Money damages?
  - [ ] Force repairs?
  - [ ] Both?
  
- [ ] Choose appropriate venue(s)
  - [ ] Conciliation Court?
  - [ ] District Court?
  - [ ] Health Department?
  - [ ] Attorney General?
  - [ ] HUD/MDHR?
  
- [ ] Check filing deadlines
- [ ] Verify jurisdiction (correct county)
- [ ] Confirm filing fees

## Filing Documents

### ✅ Required Forms
- [ ] Statement of Claim (conciliation court)
  - [ ] Fill out completely
  - [ ] Include all required information
  - [ ] Attach supporting documents
  - [ ] Make multiple copies
  
- [ ] Affidavit of Inability to Pay (if requesting fee waiver)
  - [ ] Complete financial information
  - [ ] Be honest and accurate
  - [ ] Gather income documentation
  - [ ] Gather expense documentation
  
- [ ] Summons form
  - [ ] Fill out defendant information
  - [ ] Include correct court information
  
- [ ] Any additional required local forms

### ✅ Statement of Claim Preparation
- [ ] State your name and contact info
- [ ] Identify defendant correctly
  - [ ] Full legal name
  - [ ] Correct address
  - [ ] Registered agent (if corporation)
  
- [ ] Describe what happened
  - [ ] Clear, chronological narrative
  - [ ] Stick to facts, not emotions
  - [ ] Reference specific dates
  - [ ] Cite applicable laws/lease provisions
  
- [ ] State what you want
  - [ ] Specific dollar amount
  - [ ] Itemized damages
  - [ ] Legal basis for each claim
  
- [ ] Attach exhibits
  - [ ] Label each (Exhibit A, B, C, etc.)
  - [ ] Reference in statement
  - [ ] Organize logically

### ✅ Evidence Organization
- [ ] Create evidence binder/folder
  - [ ] Table of contents
  - [ ] Tabs for each category
  - [ ] Chronological within each section
  
- [ ] Label all exhibits
  - [ ] Exhibit A, B, C, etc.
  - [ ] Brief description on each
  - [ ] Date on each
  
- [ ] Make multiple copies
  - [ ] One for you
  - [ ] One for court
  - [ ] One for defendant
  - [ ] One backup copy
  
- [ ] Bring originals to court
  - [ ] Don't give originals away
  - [ ] Show to judge if requested
  - [ ] Keep in safe place

### ✅ Filing Process
- [ ] Make copies of everything
  - [ ] At least 3 copies of all forms
  - [ ] One for court, one for defendant, one for you
  
- [ ] Go to courthouse
  - [ ] Bring all documents
  - [ ] Bring filing fee or fee waiver affidavit
  - [ ] Bring valid ID
  
- [ ] File with court clerk
  - [ ] Clerk stamps and files documents
  - [ ] Pay fee or submit waiver
  - [ ] Get file-stamped copies back
  - [ ] Get case number
  
- [ ] Get copies for service
  - [ ] Certified copies if needed
  - [ ] Enough for all defendants

## Service of Process

### ✅ Serving the Defendant
- [ ] Understand service requirements
  - [ ] Who can serve (NOT you)
  - [ ] How to serve properly
  - [ ] Time limits
  
- [ ] Choose service method
  - [ ] Sheriff service (recommended)
  - [ ] Private process server
  - [ ] Certified mail (if allowed)
  
- [ ] Provide process server with:
  - [ ] Copies of all filed documents
  - [ ] Defendant's address
  - [ ] Physical description (if helpful)
  - [ ] Best times to find defendant
  
- [ ] Get proof of service
  - [ ] Affidavit of Service
  - [ ] File with court
  - [ ] Keep copy for records
  
- [ ] Verify service deadline met
  - [ ] Usually 60 days from filing
  - [ ] Request extension if needed

## Pre-Trial Preparation

### ✅ Case Organization
- [ ] Create case summary (1-2 pages)
  - [ ] Key facts
  - [ ] Chronological overview
  - [ ] Main legal issues
  - [ ] Relief sought
  
- [ ] Prepare opening statement
  - [ ] 2-3 minutes
  - [ ] Clear and concise
  - [ ] Focus on key facts
  - [ ] Practice delivery
  
- [ ] Outline your testimony
  - [ ] What you'll say
  - [ ] Order of topics
  - [ ] Which exhibits you'll reference
  - [ ] Anticipated questions
  
- [ ] Prepare for cross-examination
  - [ ] Weak points in your case
  - [ ] How to address them
  - [ ] Stay calm and factual
  
- [ ] Prepare closing argument
  - [ ] Summarize evidence
  - [ ] Connect facts to law
  - [ ] Restate what you want
  - [ ] Practice delivery

### ✅ Witness Preparation
- [ ] Contact witnesses
  - [ ] Confirm they'll attend
  - [ ] Provide court date/time/location
  - [ ] Subpoena if necessary
  
- [ ] Prep witnesses on what to expect
  - [ ] How court works
  - [ ] Where to sit/stand
  - [ ] How to answer questions
  
- [ ] Review testimony with witnesses
  - [ ] What they'll say
  - [ ] Questions you'll ask
  - [ ] Possible cross-examination
  
- [ ] Remind witnesses of rules
  - [ ] Tell the truth
  - [ ] Don't guess
  - [ ] "I don't know" is okay
  - [ ] Stay professional

### ✅ Evidence Finalization
- [ ] Review all exhibits
  - [ ] All labeled correctly
  - [ ] All necessary documents included
  - [ ] Copies are clear/readable
  
- [ ] Create exhibit list
  - [ ] Number/letter for each
  - [ ] Description of each
  - [ ] Relevance of each
  
- [ ] Test any technology
  - [ ] If presenting digital evidence
  - [ ] Bring backup (printed copies)
  - [ ] Know how to operate equipment
  
- [ ] Prepare demonstrative aids
  - [ ] Photos enlarged if helpful
  - [ ] Timeline chart
  - [ ] Damage calculation chart
  - [ ] Property diagram/map

### ✅ Legal Arguments
- [ ] Know the elements of your claims
  - [ ] What you must prove
  - [ ] How your evidence proves each element
  
- [ ] Research defendant's likely defenses
  - [ ] How you'll counter them
  - [ ] Evidence that refutes them
  
- [ ] Cite specific laws/statutes
  - [ ] Have statute language ready
  - [ ] Know section numbers
  - [ ] Print copies for judge
  
- [ ] Know relevant case law
  - [ ] Similar cases
  - [ ] How they support your position
  - [ ] Have copies if very relevant

### ✅ Practical Preparation
- [ ] Visit courthouse beforehand
  - [ ] Know where to go
  - [ ] Find parking
  - [ ] Know security procedures
  
- [ ] Know courtroom rules
  - [ ] No phones/electronics
  - [ ] Dress code (business attire)
  - [ ] How to address judge ("Your Honor")
  
- [ ] Prepare what to bring
  - [ ] All evidence/exhibits
  - [ ] Witness list
  - [ ] Legal research
  - [ ] Notepad and pens
  - [ ] Calculator
  - [ ] Water bottle
  
- [ ] Plan arrival time
  - [ ] Arrive 30 minutes early
  - [ ] Account for traffic/parking
  - [ ] Check in with clerk

## Court Day

### ✅ Final Checklist
- [ ] Bring all documents
  - [ ] Evidence binder
  - [ ] Extra copies
  - [ ] Notes/outlines
  - [ ] Legal research
  
- [ ] Dress professionally
  - [ ] Business attire
  - [ ] Conservative colors
  - [ ] Clean and neat
  
- [ ] Bring witnesses
  - [ ] Confirm they're coming
  - [ ] Remind them of time
  - [ ] Provide transportation if needed
  
- [ ] Review case one more time
  - [ ] Opening statement
  - [ ] Key facts
  - [ ] Damage calculations
  - [ ] Closing argument

### ✅ Courtroom Behavior
- [ ] Be on time (early)
- [ ] Turn off cell phone
- [ ] Be respectful to everyone
  - [ ] Judge
  - [ ] Clerk
  - [ ] Defendant
  - [ ] Witnesses
  
- [ ] Address judge properly
  - [ ] "Your Honor"
  - [ ] Stand when speaking
  - [ ] Don't interrupt
  
- [ ] Stay professional
  - [ ] Control emotions
  - [ ] Stick to facts
  - [ ] Don't argue with defendant
  - [ ] Let evidence speak

### ✅ Presenting Your Case
- [ ] Give opening statement
  - [ ] Clear and concise
  - [ ] Set stage for your evidence
  
- [ ] Present evidence methodically
  - [ ] Follow your outline
  - [ ] Introduce each exhibit
  - [ ] Explain relevance
  
- [ ] Testify clearly
  - [ ] Speak slowly and clearly
  - [ ] Face the judge
  - [ ] Answer questions directly
  - [ ] Don't volunteer extra information
  
- [ ] Examine your witnesses
  - [ ] Ask clear questions
  - [ ] One topic at a time
  - [ ] Let them tell their story
  
- [ ] Cross-examine defendant/their witnesses
  - [ ] Point out inconsistencies
  - [ ] Ask yes/no questions
  - [ ] Don't argue
  
- [ ] Give closing argument
  - [ ] Summarize evidence
  - [ ] Connect to legal requirements
  - [ ] State what you want
  - [ ] Thank the judge

### ✅ After Trial
- [ ] Wait for judgment
  - [ ] May be immediate
  - [ ] May be mailed later (up to 30 days)
  
- [ ] Get copy of judgment
  - [ ] From clerk
  - [ ] File in safe place
  
- [ ] If you win:
  - [ ] Understand enforcement options
  - [ ] Collection procedures
  - [ ] Garnishment process
  
- [ ] If you lose:
  - [ ] Understand appeal rights
  - [ ] Time limits (usually 20 days)
  - [ ] Appeal process
  
- [ ] If settled:
  - [ ] Get agreement in writing
  - [ ] Have judge approve
  - [ ] Ensure it's enforceable

---

## Post-Judgment

### ✅ If You Win
- [ ] Review judgment carefully
  - [ ] Correct amount awarded?
  - [ ] Correct terms?
  - [ ] Any errors?
  
- [ ] Wait for payment deadline
  - [ ] Usually 20 days
  - [ ] Don't harass defendant
  
- [ ] If defendant pays
  - [ ] Get payment in full
  - [ ] Cashier's check or money order
  - [ ] File satisfaction of judgment
  
- [ ] If defendant doesn't pay
  - [ ] Wait until deadline passes
  - [ ] Begin collection process
  - [ ] Consider garnishment
  - [ ] Property liens
  - [ ] Bank levies

### ✅ Collection Process
- [ ] Get information about defendant's assets
  - [ ] Employment
  - [ ] Bank accounts
  - [ ] Property owned
  
- [ ] File for garnishment
  - [ ] Wage garnishment
  - [ ] Bank account garnishment
  
- [ ] Consider property liens
  - [ ] Real estate
  - [ ] Vehicles
  
- [ ] Keep records of all collection efforts
  - [ ] Dates of attempts
  - [ ] Amounts collected
  - [ ] Balance remaining

## Important Reminders

### Do's
- ✅ Be organized and prepared
- ✅ Stick to facts, not emotions
- ✅ Be respectful and professional
- ✅ Tell the truth
- ✅ Bring all evidence
- ✅ Dress appropriately
- ✅ Arrive early
- ✅ Listen carefully to judge
- ✅ Answer questions directly
- ✅ Follow court procedures

### Don'ts
- ❌ Don't interrupt the judge
- ❌ Don't argue with defendant
- ❌ Don't bring up irrelevant information
- ❌ Don't exaggerate or lie
- ❌ Don't use emotional language
- ❌ Don't be late
- ❌ Don't dress casually
- ❌ Don't use your phone in court
- ❌ Don't speak unless asked
- ❌ Don't give up if you lose - you can appeal
