## Evidence Organization Checklist

### Storage & Security
- [ ] All physical evidence in secure location
- [ ] Digital evidence backed up in multiple locations
- [ ] Originals separated from copies
- [ ] Important items in fireproof/waterproof container
- [ ] Password-protected digital files

### Organization
- [ ] All items numbered sequentially
- [ ] All items described in this log
- [ ] Items organized by type
- [ ] Items organized chronologically within type
- [ ] Easy to locate any item quickly

### Copies & Backups
- [ ] Copies made of all documents
- [ ] Photos backed up (cloud + external drive)
- [ ] Important items scanned to PDF
- [ ] Multiple copies of crucial evidence
- [ ] Backup copy of this log

### Court Preparation
- [ ] Evidence binder created
- [ ] Table of contents prepared
- [ ] Exhibits labeled (Exhibit A, B, C, etc.)
- [ ] Copies made for court and defendant
- [ ] Originals ready to show judge

## Important Reminders

✅ **Do's**:
- Date and label everything immediately
- Make multiple copies
- Store originals safely
- Update this log regularly
- Keep evidence organized
- Photograph everything possible
- Get witness statements in writing
- Keep receipts for everything

❌ **Don'ts**:
- Don't alter or edit evidence
- Don't give away originals
- Don't lose track of evidence
- Don't wait to document - do it now
- Don't throw anything away
- Don't assume you won't need something
- Don't let evidence degrade (protect from damage)
