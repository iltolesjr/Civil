## Timeline Presentation Tips

### For Court
- Present chronologically
- Start with lease signing
- Highlight when problems began
- Show pattern of complaints and responses (or lack thereof)
- Emphasize key dates
- Connect each event to evidence
- Show cause and effect

### Creating Visual Timeline (Optional)
Consider creating a visual timeline for court presentation:
- Use poster board or large paper
- Place key events on a horizontal line
- Use different colors for different types of events
  - Blue: Normal tenancy events
  - Red: Problems/violations
  - Green: Your complaints/reports
  - Yellow: Landlord responses
  - Orange: Official inspections/reports
