## üöÄ Getting Started

### For Case Preparation
1. Read **MASTER_ARGUMENTS_SUCCESS_GUIDE.md** for complete strategy
2. Complete the case-specific Legal Arguments guide for your case
3. Use conversation templates to document all communications
4. Run email extractor to organize email evidence
5. Print **QUICK_ARGUMENTS_CHECKLIST.md** for trial day

### For Email Organization
1. Run `run_email_extraction.bat` (Windows) or `python3 email_extractor.py` (Linux/Mac)
2. Review extracted conversations in `Extracted_Conversations/` folder
3. Use templates to properly document important communications
