# Civil

A collection of templates and tools for self-represented litigants (pro se) in civil cases.

## Trial Preparation Templates

### Pro Se Trial Preparation
- **[Pro Se Trial Preparation Template](Pro_Se_Trial_Preparation_Template.md)** - Comprehensive guide for representing yourself in court, including:
  - How to testify effectively
  - Examining friendly and hostile witnesses
  - Essential objections (hearsay, nonresponsive, speculation)
  - Getting evidence admitted (photos, audio, video, texts/social media)
  - Trial do's and don'ts
  - Closing argument preparation

### Digital Evidence
- **[Text & Social Media Evidence Template](Text_Social_Media_Evidence_Template.md)** - Complete guide for using digital communications as evidence:
  - Authentication and authorship requirements
  - Preservation methods for texts, emails, and social media
  - Platform-specific instructions (iOS, Android, Facebook, Instagram, Twitter)
  - Family law case applications
  - Court-ready formatting and presentation
